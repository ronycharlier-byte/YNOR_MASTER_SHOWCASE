# PUBLIC BRIEF YNOR

## Role
Public entry point for the corpus. This layer gives a short, stable reading path without working noise.

## Perimeter
- The public reference shows the consolidated version of the corpus, not the raw depot.
- Technical mirrors and internal artifacts stay out of scope.
- Public counts should be read as editorial inventory, not as external proof.

## Validation Level
- The mathematical layer is presented with explicit definitions, invariants, and proof chains.
- The scientific layer is organized as a validation pipeline, with some blocks already locally reproducible and others staged for targeted external review.
- The academic layer prioritizes clear hypotheses, careful wording, and stated limits.
- The strongest elements are the ones that can be reproduced locally with an explicit protocol and then handed into a structured external review package.
