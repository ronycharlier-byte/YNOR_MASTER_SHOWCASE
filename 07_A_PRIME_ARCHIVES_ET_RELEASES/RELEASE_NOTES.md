# RELEASE NOTES

## Objet
Notes de release pour les paquets terminaux du corpus.

## Release Courante
- Le corpus est stabilise autour de l'axe `A -> B -> C -> X -> C' -> B' -> A'`.
- Les couches publiques, canoniques, de soumission, de validation et d'archive sont presentes.
- La branche `A'` assume le figement final et la distribution.

## Points Notables
- Les entrees de navigation terminale sont consolidees.
- Les artefacts techniques doivent rester hors diffusion.
- Les variantes de documents doivent etre rattachees a une source canonique unique.

## Usage
Consulter ce document avant de publier, archiver ou etiqueter une version comme finale.
