# 04_X_NOYAU_MEMOIRE

## Rôle
Centre chiastique de connaissance, mémoire et intelligence.

## Entrées Principales
- `00_NODE.md` : ancrage nodal minimal
- `README.md` : entrée lisible de lecture
- `04_INDEX/README_NOYAU_MEMOIRE.md` : miroir d'index du noyau
- `07_REECRITURE_JSON_CHIASTIQUE` : couche de réécriture JSON du noyau

## Usage
Ce noyau sert de point de convergence documentaire pour les données, les miroirs, les réécritures et les artefacts de mémoire structurée.

## Position Dans L'Axe
Le noyau `X` occupe le centre de l'axe chiastique `A -> B -> C -> X -> C' -> B' -> A'`.
