# ARCHITECTURE MDL YNOR — MOTEUR AGI

Bienvenue dans l'architecture souveraine **MDL YNOR**. Ce répertoire a été réorganisé pour assurer une clarté maximale et une navigation instinctive.

## Structure du Projet

### 1. Points d'Entrée & Gestion
- **`LANCER_MOTEUR_AGI.bat`** : Le script maître pour démarrer l'audit AGI et le moteur de raisonnement.
- **`MDL_Ynor_Framework/`** : Le cœur technologique contenant tout le code source, la documentation technique et les couches théoriques.
- **`_RELEASES/`** : Dossier contenant les versions archivées, les snapshots "Phase III" et les exports compressés.
- **`_SUBMISSIONS/`** : Zone de préparation pour les soumissions académiques, incluant les lettres d'accompagnement et les listes de contrôle.

### 2. Vue d'Ensemble du Framework
Le framework est structuré de manière incrémentale :
- `_00` : Command Center & Prompts Maîtres.
- `_01` : Théories, Papiers de Recherche & LaTeX.
- `_02` : Visualisations & Graphes de Recherche.
- `_03` : Moteurs AGI & Solveurs (Navier-Stokes, etc.).
- `_04` : Déploiement, API & Guide d'Installation.
- `_07` : Administration, Légal & Gouvernance.
- `_10` : Base de Connaissance IA (Corpus Statique & Dynamique).
