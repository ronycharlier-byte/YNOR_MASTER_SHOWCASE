## Usage
Cette page sert de porte d'entree courte vers la couche des protocoles Ynor.
Les protocoles fondamentaux fixent les gestes de base.
Les protocoles secondaires declinent ces gestes par domaine.
Les protocoles par axe declinent ces gestes par grande direction Ynor.
Les protocoles sous-axes declinent ces gestes au niveau le plus fin de structuration.
Les protocoles feuilles declinent ces gestes elementaires pour les sous-axes les plus structurants.
Les protocoles sous-sous-axes donnent les gestes les plus concrets pour le travail quotidien.
Les protocoles sous-sous-sous-axes donnent les gestes les plus ultra-fins pour les noyaux operationnels.
Le noyau et sa carte miroir donnent le centre de coherence de toute cette couche.
La carte miroir des protocoles donne la boucle de travail et de controle.
