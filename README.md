# ARCHITECTURE MDL YNOR - MOTEUR AGI

Bienvenue dans l'architecture canonique **MDL YNOR**. Ce repertoire sert de point d'entree principal pour le corpus, les versions canonisees et les exports de travail.

## Structure Du Projet

### 1. Points D'entree

- **`LANCER_MOTEUR_AGI.bat`** : script maitre pour lancer l'audit AGI et le moteur de raisonnement.
- **`MDL_Ynor_Framework/`** : coeur technologique contenant le code source, la documentation technique et les couches theoriques.
- **`_RELEASES/`** : versions archivees, snapshots et exports compressees.
- **`_SUBMISSIONS/`** : zone de preparation pour les soumissions academiques et les documents de diffusion.

### 2. Organisation Fonctionnelle

Le framework est structure de maniere incrementale :

- `_00` : command center et prompts maitres.
- `_01` : theories, papiers de recherche et LaTeX.
- `_02` : visualisations et graphes de recherche.
- `_03` : moteurs AGI et solveurs.
- `_04` : deploiement, API et guide d'installation.
- `_07` : administration, legal et gouvernance.
- `_10` : base de connaissance IA.
- `_11` : geomagnetisme et WMM.
- `_12` : finance quantitative et modeles modernes.
